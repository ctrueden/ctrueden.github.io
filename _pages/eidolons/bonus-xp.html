Bonus experience

Bonus experience is XP earned from participation outside normal session gameplay.

Examples include but are not limited to:

  • Bringing snacks to game
  • Bringing soda to game
  • Bringing food to game
  • Writing a character POV for a session
  • Taking notes for a session

Regarding length: even a single paragraph is OK, though we encourage more substantial musings rather than very short blurbs. We may cease granting bonus XP if players repeatedly submit borderline session entries.

It is fine to "retroactively" write session notes for any prior session. We will grant up to one block of bonus XP per session. These blocks start small, and increase with each contribution, so it is increasingly valuable to participate. More importantly, we as GMs, as well as the other players, find it immensely valuable to read character perspectives, as it helps us to better understand character motivations, and provide compelling plot points.
