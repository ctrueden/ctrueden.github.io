Blighter

The blighter prestige class (Complete Divine p. 23-25) has the following changes to the requirements:

  • Requires 2nd level druid spells
  • No BAB requirement
  • Approved through 5th level for the moment

Note the following changes and clarifications to the blighter class features:

  • If you deforest a nexus, you are good for a month instead of 24 hours, but the nexus will be corrupted.
  • Blightfire costs 5 vitality to use.
  • Sustenance is only active when deforestation has been fulfilled.
  • Do not use the Blighter spells per day table. Instead, add your blighter level and ex-druid level, and use the druid spells known table, but with the blighter spell list instead.

The following changes are in effect for the blighter spell list:

  • Replace animate dead with animate dead animal, as the 6th level blighter ability.
  • Decomposition works with wound damage only.
  • Remove languor from the spell list.

For undead wild shape:

  • Type changes to undead
  • No natural armor bonus, except as listed in the appropriate beast shape spell
  • No stat changes, except as listed in the appropriate beast shape spell
  • Constitution score is treated as dash, except that there is no change to vitality or wound score
  • Immunity to cold
  • Damage reduction 5/bludgeoning
  • At 5th level, use beast shape II
  • At 9th level, use beast shape III
