Chameleon

See Races of Destiny p. 111, with the following changes:

  • spellcraft skill requirement is geomancy instead
  • able learner feat requirement is waived
  • must be kitsune race
  • can only change aptitude focus when A) exposed as fraud or B) succeeds in breeding or C) other extenuating circumstances where Spider would forget his prior persona
  • for spells known, use the table corresponding to the class being emulated, with an effective class level equal to the chameleon level plus half non-chameleon levels (no spellpool or other class benefits)
