Gestalt

One of the options possible with the merit system is to play a gestalt character.

Naively, the math for base attack and saving throw progression can be abused for gestalt characters. If you are using the gestalt option, please note that we will be using a fractional progression for base attack and saves to address this.

Also note that other overlapping class features with a standard progression will be handled in this fashion. Examples:

  1. If you gestalt runemage and also take a prestige class that gives +1 arcane caster level, you will gain only one runemage caster level per level, not two.
  2. Gestalt rogue with ninja will advance your sneak attack / sudden strike at the normal rate (+1 die every other level), not twice as fast.

Please use common sense, and ask if you are unsure. There will certainly be edge cases that come up, which we'll need to rule on, so don't be afraid to ask if it's applicable to your character. If this stuff doesn't make sense, don't worry about it, we can clarify during the character creation session.
